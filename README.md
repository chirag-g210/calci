## 🧮 Flask Calculator

A modern web-based calculator built using Python (Flask) for the backend and HTML, CSS, JavaScript for the frontend.

It performs basic arithmetic operations with a clean neumorphic UI and real-time calculations via a Flask API.

---

## 🚀 Features

➕ Addition, ➖ Subtraction, ✖ Multiplication, ➗ Division

🧹 AC (All Clear) button

⌫ DEL (Delete last digit)

📱 Responsive calculator layout

🎨 Neumorphic dark UI design

⚡ Real-time calculation using Flask backend

🔌 Frontend ↔ Backend API integration using fetch()

## ⚙️ Installation & Setup

1️⃣ Clone the repository
git clone https://github.com/chirag-g210/calci.git
cd calci

2️⃣ Create virtual environment (recommended)
python -m venv venv

Activate:

Windows
venv\Scripts\activate

3️⃣ Install dependencies
pip install flask

4️⃣ Run the application
python main.py

5️⃣ Open in browser
http://127.0.0.1:5000
